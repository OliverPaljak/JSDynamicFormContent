Constructor

Create new form content object
Parameters:
Name Type Description
formElement HTMLFormElement Single form element
inputSelectors string Selectors for elements which will be used to read and display data (like jQuery selectors: # for IDs, . for classes and so on). Separate multiple selectors with comas.
submitButton HTMLButtonElement | HTMLInputElement Submit button to display form data (although the type should be always 'button')
